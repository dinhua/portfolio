2. 字型樣式


粗體

使用 class 屬性搭配 font-bold 包住需要加粗的字段，例如：

我是一段需要加粗的文字

程式碼

<p class="font-bold">我是一段需要加粗的文字</p>


斜體

使用 class 屬性搭配 font-italic 包住需要傾斜的字段，例如：

我是一段需要傾斜的文字

程式碼

<p class="font-italic">我是一段需要傾斜的文字</p>


底線

使用 class 屬性搭配 text-underline 包住需要底線的字段，例如：

我是一段需要底線的文字

程式碼

<p class="text-underline">我是一段需要底線的文字</p>


文字對齊

使用 class 屬性搭配 text-left, text-center, text-right 改變整段文字的對齊方式，例如：

置左對齊

置中對齊

置右對齊

程式碼

<p class="text-left">置左對齊</p>
<p class="text-center">置中對齊</p>
<p class="text-right">置右對齊</p>


你一定會問的小訣竅！


如何同時使用多種樣式？

在使用 class 屬性搭配多種樣式時，使用 (空格) 隔開，例如：

我是一段又粗又斜的文字

程式碼

<p class="font-bold font-italic">
　我是一段又粗又斜的文字
</p>


如何在段落中局部使用樣式？

在段落中使用 <span></span> 標籤搭配 class 屬性，可以讓局部文字產生行內效果，例如：

我是一段又粗又斜的文字

程式碼

<p>
　我是一段<span class="font-bold">又粗</span><span class="font-italic">又斜</span>的文字
</p>
